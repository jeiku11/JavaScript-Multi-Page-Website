Simple CSS Quiz

Test your knowledge in CSS.

1. Which of the following is the correct way to apply an external CSS file to an HTML document?

b. <script>
c. <body>
d. <link>

2. What does the z-index property in CSS control?

3. Which CSS property is used to make text bold?

a. font-style
b. text-decoration
c. font-weight
d. font-bold

4. Which of the following is NOT a valid CSS value for the position property?

a. static
b. relative
c. fixed
d. center

5. What does the flex-grow property do in a flexbox container?

a. Defines how much an item should grow relative to others
b. Controls the alignment of items
c. Sets the order of the flex items
